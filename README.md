# qnze-gamepad
Arduino gamepod

qnze-gamepod Arduino nano gamepod &amp; software<br>
テトリスを遊びやすくするためにボタンとスピーカーを２つ追加。<br>
ステレオで効果音がでます。<br>
電池ボックスもオプションで追加できる。<br>


<img src=./images/aq-gamepod1_ブレッドボード.jpg height=600><br>

接続 <br>
D8 Speaker1<br>
D9 Speaker2<br>
D2 Shoot button<br>
A0 Left button<br>
A1 Up button<br>
A2 Down button<br>
A3 Right button<br>
A4 OLED SDA<br>
A5 OLED SCA<br>
5V OLED VCC<br>
GND OLED GND<br>
<br>
２つの音が同時に出ています。<br>
